📚 How to Install Bookmarklets

Bookmarklets are small JavaScript programs that run when you click a bookmark. Simply drag the buttons below to your bookmark bar to install them.

💡 Quick Install: Drag any blue button directly to your browser's bookmark bar to install it instantly! Alternatively, click "Copy Code" to get the JavaScript code for manual installation.

🌐 Chrome / Edge

  1. Make sure your bookmark bar is visible (Ctrl+Shift+B)
  2. Drag the blue button to your bookmark bar
  3. The bookmarklet is now ready to use
  4. Click it on any website to activate

🦊 Firefox

  1. Show your bookmarks toolbar (Ctrl+Shift+B)
  2. Drag the blue button to the toolbar
  3. The bookmarklet will be automatically installed
  4. Click to run on any webpage

🧭 Safari

  1. Show the bookmarks bar (View → Show Favorites Bar)
  2. Drag the blue button to the bar
  3. Safari will automatically create the bookmark
  4. Click to execute on any site
